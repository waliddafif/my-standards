Cree une nouvelle page ou composant frontend.

## Processus

1. **Explorer** — verifier les composants, hooks et utils existants avant de creer
2. **Clarifier** — poser des questions sur le design/comportement attendu
3. **Planifier** — lister les fichiers, presenter pour validation

4. **Executer** → deleguer a l'agent `frontend-feature`

5. **Verifier** :
   - `npm run build` pour les erreurs de type
   - `browser_snapshot` pour verifier le rendu (pas de screenshot)

## Regles

- Reutiliser les composants UI existants (shadcn, etc.)
- Texte utilisateur en francais, code en anglais
- Pas de secrets dans les Client Components
- Chercher les composants similaires avant d'en creer un nouveau
